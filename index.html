Welcome to the CI/CD Demo Site

This tiny site exists for one reason: to get pushed to Git and watched by a pipeline.

What happens when you push a change?

  1. Code gets checked out on a fresh runner
  2. HTML gets linted (syntax check, like ansible-playbook --syntax-check)
  3. Files get scanned for hardcoded secrets/passwords
  4. A "build" artifact gets produced and archived
